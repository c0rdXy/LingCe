# Changelog

## V0.1.3

- 修复练习范围和题型重复点击导致练习统计刷新的问题。
- 优化错题复习逻辑，清空错题后不会再进入残留错题范围。
- 修复错题复习中题型筛选和“全部题型”切换失效的问题。
- 梳理 `settings.json` 与 `user_data.json` 的职责边界，减少重复配置。
- 调整首页功能入口层级，突出练习模式和考试模式，弱化辅助工具按钮。

## V0.1.2

- 优化 AI 设置页的接入方式、服务商、Base URL、模型和 Key 切换逻辑。
- 支持按接入方式与服务商分组保存多个 API Key，并在配置文件中加密存储。
- 新增获取可用模型功能，切换服务商后可按当前 Key 拉取模型列表。
- 优化 AI 复核助手回答展示，提升长内容和 Markdown 文本的阅读体验。
- 完善 Coding Plan 与 Token Plan 常用服务商预设地址。

## V0.1.0

- 新增 AI 复核设置，支持模型 API、模型 Coding Plan 和自定义模型三类接入方式。
- 新增 DeepSeek、Kimi、GLM、MiniMax、GPT、Gemini、小米 MiMo 等常见模型预设。
- 练习模式和考试回顾模式新增“AI 复核本题”入口，支持围绕当前题目继续追问。
- 题库生成器新增 AI 复核当前题和 AI 生成答案解析能力。
- AI 复核助手改为左右聊天界面，并支持基础 Markdown 渲染。
- 新增 AI 服务层、连接测试和连接耗时显示。

## V0.0.10

- 清理项目中不再使用的旧组件和残留工具函数。
- 删除无引用的统计图表占位文件。
- 统一代码注释表述，去除开发过程痕迹。
- 完善 README 与更新日志中的文件说明。

## V0.0.9

- 优化系统并解决一些已知 BUG。
- 新增 `question_banks/` 题库目录，将题库文件与 `data/` 运行数据分离。
- 完善手工题库生成器的保存路径、未完成题提示、校验定位和关闭提醒。
- 统一判断题答案显示与保存格式为“正确/错误”，并兼容 A/B 等常见答案格式。

## V0.0.8

- 解决练习范围返回首页后未正确保持的问题。
- 优化“继续上次”和练习范围选择的状态保存逻辑。
- 恢复重新进入练习模式后的错题复习历史。
- 首页新增 GitHub 图标和 Star 提示。
- 清理无效代码和说明文本。

## V0.0.7

- 新增发布检查脚本，用于版本、运行数据、文件大小、乱码、编译和测试检查。
