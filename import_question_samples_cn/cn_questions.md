# 中文五题型题库示例

用于测试 LingCe 的 AI 解析导入功能。

## 第 1 题

**题型：** 单选题
**Type:** single

**题干：** Python 中用于定义函数的关键字是哪一个？

- A. class
- B. def
- C. return
- D. import

**答案：** B
**解析：** Python 使用 def 关键字定义函数。

## 第 2 题

**题型：** 多选题
**Type:** multiple

**题干：** 下列哪些属于常见的关系型数据库？

- A. MySQL
- B. PostgreSQL
- C. Redis
- D. SQLite

**答案：** ABD
**解析：** MySQL、PostgreSQL、SQLite 是关系型数据库；Redis 通常是键值型内存数据库。

## 第 3 题

**题型：** 判断题
**Type:** judgement

**题干：** HTTP 状态码 404 通常表示服务器内部错误。


**答案：** 错误
**解析：** 404 表示资源未找到；服务器内部错误通常是 500。

## 第 4 题

**题型：** 填空题
**Type:** fill

**题干：** 在 Git 中，用于查看当前工作区状态的命令是 git ____。


**答案：** status
**解析：** git status 可查看暂存区、工作区和分支状态。

## 第 5 题

**题型：** 简答题
**Type:** short

**题干：** 请简述单元测试的主要作用。


**答案：** 单元测试用于验证最小代码单元的行为是否符合预期，帮助尽早发现回归问题，并提升代码修改时的信心。
**解析：** 答案应包含验证功能、发现问题、降低回归风险等要点。

## 第 6 题

**题型：** 单选题
**Type:** single

**题干：** 在计算机网络中，DNS 的主要作用是什么？

- A. 加密网络通信
- B. 把域名解析为 IP 地址
- C. 压缩网页内容
- D. 检测硬盘坏道

**答案：** B
**解析：** DNS 负责域名和 IP 地址之间的解析。

## 第 7 题

**题型：** 多选题
**Type:** multiple

**题干：** 下列哪些做法有助于保护账号安全？

- A. 启用双因素认证
- B. 多个网站使用同一个简单密码
- C. 定期更新密码
- D. 不随意点击陌生链接

**答案：** ACD
**解析：** 双因素认证、定期更新密码、不点击陌生链接都有助于安全；复用简单密码风险较高。
